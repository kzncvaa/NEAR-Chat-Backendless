1. Open a Terminal window, and go to the project's directory.
2. Run the pod install / pod update command
Once all of the pod data is downloaded / updated, the .xcworkspace file will be created.
3. Open the .xcworkspace file to launch your project and build it.

**************************************************
You need the real iOS device to run this code, because the example includes camera usage.
**************************************************

1. Press "Take a Photo" to create the FileObject table in the Backendless console and create a photo that will be uploaded into the file service and linked to the created table.
2. Press "Browse Uploaded" to get the list of uploaded photos.
