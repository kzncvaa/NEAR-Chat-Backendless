1. Open a Terminal window, and go to the project's directory.
2. Run the pod install / pod update command
Once all of the pod data is downloaded / updated, the .xcworkspace file will be created.
3. Open the .xcworkspace file to launch your project and build it.
4. Follow these steps to create .p12 certificate for iOS push notifications and Backendless/backend configuration: https://backendless.com/docs/ios/push_push_notification_setup_ios.html.
5. Select your development team (signing is necessary for push notifications).
6. Build the project.

**************************************************
You need the real iOS device to run this code.
